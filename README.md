# 🥗🍎 Calorie and Nutrition Tracker System 🍎🥗

The **Calorie and Nutrition Tracker System** is a C++ based mini project developed to support the United Nations Sustainable Development Goal of **Zero Hunger** 🌍. This project focuses on promoting healthy eating habits, nutrition awareness, and proper food management through technology.

In today’s world, many people suffer from malnutrition, unhealthy eating habits, and lack of nutritional awareness. This system helps users monitor their daily food intake by tracking important nutritional values such as calories, proteins, carbohydrates, and fats. The application provides a simple and user-friendly way to maintain a balanced diet and encourage healthier lifestyles.

💡 The main objective of this project is to:

* Promote healthy nutrition habits
* Create awareness about balanced diets
* Help users monitor calorie intake
* Support hunger management and food planning
* Reduce nutritional deficiencies

✨ **Key Features of the System**

* ➕ Add daily food items
* 🍽️ Track calorie intake
* 💪 Monitor protein consumption
* 🍞 Calculate carbohydrates
* 🥑 Measure fat intake
* 📋 View complete nutrition records
* 📊 Display total nutrition summary
* 🖥️ Simple menu-driven interface

🛠️ **Technologies Used**

* 💻 C++ Programming Language
* 🧠 Object-Oriented Programming (OOP)
* 📚 Arrays and Functions
* 🔄 Loops and Conditional Statements

📖 The system allows users to enter food details and automatically calculates the total nutritional intake. It acts as a digital nutrition assistant that helps users understand the importance of balanced meals and healthy food consumption.

🌟 **Advantages of this Project**

* Easy to use for beginners
* Helps maintain healthy eating habits
* Encourages nutrition awareness
* Reduces chances of unhealthy diets
* Supports smart food management
* Useful for students, gyms, hospitals, and diet planners
